API zadanie

API slovenských dátumov. Funkcionalita:
  • Vyhľadanie menín na základe dátumu
  • Vyhľadanie dátumu na základe mena a štátu
  • Výpis štátnych sviatkov na Slovensku
  • Výpis štátnych sviatkov v Českej republike
  • Výpis pamätných dní na Slovensku
  • Pridanie mena k určenému dátumu

GET name by date

http://147.175.98.72/Zadanie6/search/?date=dateValue

GET date by name&state

http://147.175.98.72/Zadanie6/search/?name=nameValue&&state=stateValue

GET holiday SK

http://147.175.98.72/Zadanie6/holidaySK

GET memorial days SK

http://147.175.98.72/Zadanie6/memorialDaySK

GET holiday CZ

http://147.175.98.72/Zadanie6/holidayCZ

POST new name

http://147.175.98.72/Zadanie6/addName
